## Что это

Приложение для мамы. Она работает по сменному графику и ей нужно быстро спрогнозировать на дни вперёд -- работает она или нет. Это презентационный проект, а не продукт для широкого использования.

Цикл из 4 фаз, который идёт нон-стопом:
- **День** -- дневная смена
- **Ночь** -- ночная смена
- **С ночи** -- отдых после ночной
- **Отдых** -- выходной

Достаточно один раз указать фазу для любого дня -- и весь календарь рассчитается автоматически, вперёд и назад.

## Установка

Открыть в Chrome на Android, нажать **"Добавить на главный экран"**.
